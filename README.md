# Guitar Chord Trainer

一个纯前端吉他和弦练习器原型：选择 4-8 个和弦后进入练习页，页面只显示当前和弦与下一个和弦，并通过浏览器麦克风检测是否弹中了目标和弦。

在线地址：https://guitar-chord-trainer.pages.dev/

## 运行

```powershell
git clone https://github.com/eddietsai6-code/guitar-chord-trainer.git
cd guitar-chord-trainer
python -m http.server 8027
```

然后打开：

```text
http://127.0.0.1:8027/
```

## 使用

1. 在选择页搜索或筛选和弦。
2. 选择任意 4-8 个和弦。
3. 选择是否显示标准指法图。
4. 点击“开始练习”。
5. 浏览器请求麦克风权限时选择允许。
6. 按目标和弦扫弦；识别正确后立即通过并进入下一题。

## 识别原理

页面使用 Web Audio API 读取麦克风频谱，把 70-1400 Hz 范围内的能量聚合到 12 个音级。它会先寻找一次扫弦的能量起点，再收集约 180ms 的短窗口，把窗口内的频谱平均后评分。判断时只在你本轮选择的 4-8 个和弦里竞争：目标和弦必须是第一名，并且要和第二名拉开差距，才会通过。

这不是“按住保持 0.6 秒”的逻辑。你扫完一次后，系统会在短窗口结束时判断；如果正确就进入下一题。通过后仍然保留短冷却，避免同一次扫弦连续跳过多题。

这不是机器学习模型，优点是本地运行、响应快、不上传声音；缺点是对环境噪声、扫弦力度、变调夹、复杂转位和过强泛音会比较敏感。后续可以继续加入更细的基频追踪、字符串权重、误触过滤和校准流程。

指法图覆盖全部 132 个和弦。常见开放和弦使用手写标准形，其他和弦使用可移动 E/A 根音把位；高把位只显示类似 `5f` 的起始把位提示。
